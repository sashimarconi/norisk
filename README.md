# norisk

Site estatico pronto para deploy na Vercel.

## Publicacao na Vercel

1. Importe este repositorio no painel da Vercel.
2. Framework Preset: `Other`.
3. Build Command: vazio.
4. Output Directory: vazio.
5. Deploy.

## Estrutura

- Entrada principal em `index.html`.
- Pagina principal do espelho em `importscap.shop/import/index.html`.
